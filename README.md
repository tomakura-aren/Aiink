# COEIRO-GPT
ChatGPTとVOICEVOXやCOEIROINKを連携し、キャラクターと音声で会話するソフトウェアです。

# 注意事項
- 素人コーディングのため、無駄な動作やコードが含まれる場合があります。
  (まあそもそもPythonで作るプログラムでもないけどね...)
- Windows 64bit以外では動きません！(Macが手に入ればMac版も出します...)
